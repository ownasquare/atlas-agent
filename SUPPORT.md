# Support

Atlas is a community-maintained local application. Use this order when you need help:

1. Follow [Getting started](docs/getting-started.md).
2. Run `uv run atlas doctor` and follow its next action.
3. Check [Troubleshooting](docs/troubleshooting.md).

Once this repository is published, use its GitHub issue templates for reproducible bugs and focused
feature requests. Do not include API keys, private prompts, private files, personal paths, or user
data. No public issue URL is claimed before publication.

Report vulnerabilities and sensitive security findings through the private process in
[SECURITY.md](SECURITY.md), not through a public issue.
